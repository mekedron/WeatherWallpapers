INSIDE THE APP

Built like a native app,
because it is one.

A clean SwiftUI app across iPhone, iPad and Mac — everything lives on your devices.

▦
LIBRARY
Your wallpaper library

Every set in one grid — source art, target device and live generation progress at a glance. Import or export any set as a ZIP.

◱
SET DETAIL
The 96-variant grid

All 96 images grouped by weather and time of day. Regenerate a single image, a whole row, or the full set — with per-set cost and token tracking.

✦
NEW SET WIZARD
A guided three-step flow

Name your set, pick a built-in or custom device resolution, then choose a photo, a file, or generate the source artwork with AI.

⚙
SETTINGS
Keys, spending & more

Store provider API keys in the Keychain, review spending, browse weather stats, and switch the interface language — no backend involved.

FOUR STEPS

Set it once. Forget it forever.

01
Create a set

Pick a photo, choose a file, or generate the source artwork with AI from any prompt.

02
Pick your device

iPhone, iPad or MacBook resolutions built in — or add a custom size of your own.

03
Generate 96

The app re-lights your image into every weather and time of day. Regenerate any you like.

04
Automate

The “Get Current Wallpaper” Shortcut reads your weather and sets the matching image.

Shortcut: 🖼 Get Current Wallpaper → ⚙ Set Wallpaper

PRIVACY

Nothing leaves your devices.

✓No backend, no analytics, no accounts.
✓API keys live in the Keychain, sent only to the provider you chose.
✓Location never leaves the device — used only for the weather request.
✓Sets are plain folders of PNGs in your iCloud Drive — usable without the app.
